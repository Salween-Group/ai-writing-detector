# AI Writing Detector

## What this skill does

This skill scores a piece of writing on a 0-100 scale for how AI-generated it sounds. 100 is human-sounding. It auto-detects the content type, runs through 40+ AI writing patterns (universal + channel-specific), checks an expanded banned vocabulary list, assesses originality, and delivers a single score with a detailed diagnostic report.

This is a diagnostic tool, not a rewriter. It identifies the problems and shows what to fix. 

Inspired by some code generously shared here: https://github.com/ericosiu/ai-marketing-skills and a LinkedIn post by Aditya Vempaty.
